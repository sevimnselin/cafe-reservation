# cafe-reservation
A cafe reservation website with an interactive booking form, built with HTML, CSS, and JavaScript.

## Features
- Responsive layout for desktop and mobile  
- Reservation form with validation  
- Confirmation message displayed after form submission  
- Clean, user-friendly interface  

## Tech Stack
- HTML5  
- CSS3  
- JavaScript

https://sevimnselin.github.io/cafe-reservation
